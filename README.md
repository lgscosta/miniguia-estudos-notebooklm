# miniguia-estudos-notebooklm


## 1. Contexto e Objetivos
Esse projeto visa aplicar conhecimentos adiquiridos sobre o uso do NotebookLM de forma prática através de um assunto de interesse. O assunto escolhido foi a otimização de performance e queima de gordura via Protocolo Tabata.

**Objetivo:** 
* Compreender a origem científica do protocolo (Estudo de 1996).
* Diferenciar o "Tabata Real" de variações comuns de fitness.
* Identificar os mecanismos de queima de gordura pós-treino (EPOC).


## 2. Curadoria de Fontes
Para alimentar o NotebookLM, selecionei as seguintes fontes de alta credibilidade:

1. **[Estudo Original]** [Analysis of Tabata Training - PubMed](https://pubmed.ncbi.nlm.nih.gov/8897392/): O pilar científico de 1996 que validou a eficiência do treinamento intermitente de alta intensidade.
2. **[Perfil do Autor]** [Dr. Izumi Tabata no ResearchGate](https://www.researchgate.net/profile/Izumi-Tabata): Repositório central onde foram extraídos diversos PDFs e artigos complementares para utilizar como fontes de dados.
3. **[Base Científica]** [Scientific bases for the superiority of the Tabata training](https://www.researchgate.net/publication/361666915_Scientific_bases_for_the_superiority_of_the_Tabata_training): Artigo de 2022 que explica mecanicamente por que o método supera outros tipos de treinamento em termos de capacidade aeróbica e anaeróbica.
4. **[Perspectiva Histórica]** [History of Tabata training](https://www.researchgate.net/publication/361681792_History_of_Tabata_training): Documento essencial para entender como o protocolo nasceu na patinação de velocidade e se tornou um fenômeno mundial no fitness.
